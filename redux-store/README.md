## Prerequisites

* Install [Node.js](https://nodejs.org/en/download/)

* Install TypeScript

```bash
npm install -g typescript
```

## Setup

#### Installing dependencies

Changing package versions is not recommended due to issue with newer ones

```
npm install
```

#### Local server

```npm start```
